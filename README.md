# Employee-Attrition-Analysis

⭐ Key Project for Data Analyst Portfolio

# Overview

This project analyzes employee data to understand attrition patterns and identify key factors affecting employee retention. It focuses on workforce trends, job satisfaction, salary, and work-life balance to derive actionable HR insights.

# Objective

To analyze employee attrition and identify the key drivers influencing employee turnover, and provide data-driven recommendations to improve retention and organizational performance.

# Tools & Technologies

* Python (Pandas)
* SQL
* Power BI

# Dataset

The dataset contains 400 employee records with attributes such as department, salary, experience, job satisfaction, work-life balance, and attrition status. 

# Steps Performed

* Data cleaning and preprocessing using Python (Pandas)
* Handled duplicates and validated dataset consistency
* Feature engineering (Salary groups and Experience groups)
* Performed SQL analysis to calculate attrition rate and identify patterns
* Analyzed attrition across departments, salary groups, experience levels, and satisfaction
* Built interactive Power BI dashboard to visualize workforce trends

# Key Insights

* Overall attrition rate is approximately 47.75% indicating high employee turnover 
* IT and Sales departments show the highest attrition
* Employees with low salary show the highest attrition rates
* Poor work-life balance is strongly linked to higher attrition
* Mid-experience employees (6–10 years) show significant attrition
* Low job satisfaction directly impacts employee retention

# Business Impact

This analysis helps organizations identify key drivers of employee attrition and implement targeted strategies such as improving compensation, enhancing work-life balance, and creating better career growth opportunities.

# Dashboard

Power BI dashboard showcasing attrition rate, department-wise analysis, salary impact, and employee satisfaction trends.

# Recommendations

* Improve compensation and benefits for low-salary employees, as salary is a key factor influencing attrition. This aligns with Maslow’s hierarchy, where financial security is a basic need.

* Enhance work-life balance through flexible policies and better workload management, supporting employee well-being and long-term retention.

* Focus on mid-experience employees (6–10 years) by providing clear career growth opportunities and skill development programs, addressing higher-level needs like esteem and self-actualization (Maslow).

* Apply Fayol’s principles such as proper division of work, clear authority and responsibility, and better communication to improve organizational structure and employee satisfaction.

* Implement department-specific strategies, especially for IT and Sales, to address role-specific challenges and reduce attrition.
